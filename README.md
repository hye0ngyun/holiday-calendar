# open api 공휴일 데이터를 활용한 캘린더

## 사용 기술

### Frameword(library)

react.js

### Utils

dayjs

### UI

tailwindcss

### DevOps(hosting)

gh-pages

## 요구사항

- UI 스크린샷 참고하여 디자인 구현
- 우측 상단 `<`, `오늘`, `>` 버튼 클릭시 각각 이전 달, 현재 달, 이후 달로 변경
- [공공데이터 포털의 특일정보 API](https://www.data.go.kr/data/15012690/openapi.do) 활용하여 공휴일 표시
- 특정 날짜 클릭 시 스케쥴 입력, 조회, 수정, 삭제 가능 (CRUD)
